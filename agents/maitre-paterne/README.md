# Maître Paterne – Agent émergent

Cet agent analyse une image, extrait chaque détail, cherche motifs et différences, puis génère un message crypté/signé à la Trinity. Le message est prêt pour log/fusion dans Supabase ou workflow agentic.

## Usage

```bash
python analyse_image.py votre_image.jpg
```

## Fonction
- Extraction taille, couleurs, dominante
- Description/feedback pattern
- Structuration universelle Trinity (message JSON + signature HMAC)
- Adaptable à l’intégration IA/Trinity/Comet
